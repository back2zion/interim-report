6단계 자동화 프로세스

1
📱

모바일 앱 파일 전송

사용자가 모바일 앱에서 녹음파일을 ex-GPT 시스템으로 직접 전송하여 프로세스를 시작합니다.

지원 형식: WAV, MP3, M4A
최대 크기: 100MB
전송 방식: HTTP Upload
2
🌐

ex-GPT 백엔드 수신

ex-GPT 백엔드 시스템에서 업로드된 파일을 수신하고 처리를 위한 준비 작업을 수행합니다.

프레임워크: FastAPI
파일 검증: 형식/크기 확인
임시 저장: 로컬 스토리지
3
🎤

Whisper Large V3 STT

OpenAI Whisper Large V3 모델을 활용하여 음성 파일을 텍스트로 변환합니다.

모델: Whisper Large V3
한국어 CER: 3.1%
처리 속도: 실시간 대비 0.3x
4
📚

사전 활용 후처리

도로공사 전문용어 사전을 활용하여 STT 결과를 보정하고 정제합니다.

방식: 사전 기반 매칭
전문용어: 하이패스, IC, 휴게소
처리: 문자열 치환
5
🤖

LLM 회의록 생성

대화형 AI 모델을 활용하여 STT 텍스트를 구조화된 회의록 형태로 변환합니다.

모델: Qwen3-235B-A22B
템플릿: 도로공사 표준양식
구조화: 주요 안건/결정사항
6
📧

이메일 자동 발송

처리 완료 후 STT 텍스트 파일과 회의록 파일을 각각 txt 형태로 이메일 첨부하여 발송합니다.

첨부파일: 2개 txt 파일
발송 방식: SMTP 자동발송
처리 시간: 5-10분
